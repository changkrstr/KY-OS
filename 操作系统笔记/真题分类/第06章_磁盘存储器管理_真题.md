### 【2018年第8题】磁臂粘着

**来源**：2018年全国统考

**题目**：

8.系统总是访问磁盘的某个磁道而不响应对其他磁道的访问请求，这种现象称为磁臂粘着。下列磁盘调度算法中，不会导致磁臂粘着的是（   ）。
　　A.先来先服务（FCFS）        B.最短寻道时间优先（SSTF）
　　C.扫描算法（SCAN）           D.循环扫描算法（CSCAN）

**参考答案**：A

**解析**：当系统总是持续出现某个磁道的访问请求时，SSTF、SCAN和CSCAN均会持续满足最短寻道/扫描条件，一直服务该访问请求，从而导致磁臂粘着。**先来先服务（FCFS）按照请求次序进行调度，比较公平**，不会导致磁臂粘着。

### 【2018年第9题】文件访问速度优化

**来源**：2018年全国统考

**题目**：

9.下列优化方法中，可以提高文件访问速度的是（   ）。
　　I.提前读                II.为文件分配连续的簇
　　III.延迟写                IV.采用磁盘高速缓存
　　A.仅I、II        B.仅II、III        C.仅I、III、IV        D.I、II、III、IV

**参考答案**：D

**解析**：
- I. **提前读**：在读当前盘块时，将下一个可能要访问的盘块数据读入缓冲区，提高文件访问速度
- II. **为文件分配连续的簇**：减少寻道时间，提高文件访问速度
- III. **延迟写**：先将写数据写入缓冲区，减少了访问磁盘的次数，提高文件访问速度
- IV. **采用磁盘高速缓存**：频繁使用的磁盘数据暂存在高速缓存中，减少访问磁盘的次数

以上四种方法均可以提高文件访问速度。本题选D。

### 【2015年第11题】SCAN磁盘调度

**来源**：2015年全国统考

**题目**：

11.某硬盘有200个磁道（最外侧磁道号为0），磁道访问请求序列为130，42，180，15，199，当前磁头位于第58号磁道并从外侧向内侧移动。按照SCAN调度方法处理完上述请求后，磁头移过的磁道数是（   ）。
　　A.208      B.287      C.325      D.382

**参考答案**：C

**解析**：SCAN为电梯调度算法。磁头当前位于58号磁道，向内侧（磁道号增大的方向）移动：

磁头移动路径：58 → 130 → 180 → 199 → 15 → 42

磁头移过的磁道数 = (130-58) + (180-130) + (199-180) + (199-15) = 72 + 50 + 19 + 184 = **325**

（注：SCAN到达最内侧磁道199后，转向向最外侧移动，到达15号磁道。由于请求中还有42号磁道，在转向途中经过42号磁道时处理。）

### 【2013年第3题】磁盘I/O设备驱动程序

**来源**：2013年全国统考

**题目**：

3.用户程序发出磁盘I/O请求后，系统的处理流程是：用户程序→系统调用处理程序→设备驱动程序→中断处理程序。其中，计算数据所在磁盘的柱面号、磁头号、扇区号的程序是（   ）。
　　A.用户程序    B.系统调用处理程序    C.设备驱动程序    D.中断处理程序

**参考答案**：C

**解析**：计算磁盘号、磁头号和扇区号的工作是由**设备驱动程序**完成的。设备驱动程序接收来自上层的抽象I/O请求，将其转换为具体的磁盘物理地址（柱面号、磁头号、扇区号），然后启动磁盘控制器执行实际的I/O操作。

### 【2012年第10题】磁盘I/O性能

**来源**：2012年全国统考

**题目**：

10.下列选项中，不能改善磁盘设备I/O性能的是（   ）。
　　A.重排I/O请求次序       B.在一个磁盘上设置多个分区
　　C.预读和滞后写           D.优化文件物理块的分布

**参考答案**：B

**解析**：
- A. **重排I/O请求次序**：按照磁盘调度算法重新排列I/O请求，可以减少寻道时间，**能改善**I/O性能
- B. **在一个磁盘上设置多个分区**：分区只是逻辑划分，**不能改善**磁盘I/O性能
- C. **预读和滞后写**：减少磁盘访问次数，**能改善**I/O性能
- D. **优化文件物理块的分布**：减少寻道时间，**能改善**I/O性能

本题选B。

### 【2011年第9题】单缓冲/双缓冲计算

**来源**：2011年全国统考

**题目**：

9.某文件占10个磁盘块，现要把该文件磁盘块逐个读入主存缓冲区，并送用户区进行分析， 假设一个缓冲区与一个磁盘块大小相同，把一个磁盘块读入缓冲区的时间为100us，将缓冲区的数据传送到用户区的时间是50us，CPU 对一块数据进行分析的时间为50us。在单缓冲区和双缓冲区结构下，读入并分析完该文件的时间分别是（   ）。
　　A.1500us、1000us   B.1550us、1550us   C.1550us、1100us   D.2000us、2000us

**参考答案**：C

**解析**：

**单缓冲区**：当上一个磁盘块从缓冲区读入用户区完成时，下一磁盘块才能开始读入。
- 传输一个数据块的总时间 = 读入缓冲区100us + 传送到用户区50us = 150us
- 10个数据块的传输时间 = 150 x 10 = 1500us
- 加上最后一个数据块的CPU分析时间50us
- **总时间 = 1500 + 50 = 1550us**

**双缓冲区**：两个缓冲区交替使用，缓冲区读入用户区时可以同时读入下一个磁盘块到另一个缓冲区。
- 前9个磁盘块可以连续读入缓冲区，CPU可同时分析：9 x 100us = 900us
- 最后一个磁盘块读入缓冲区100us + 传送到用户区50us + CPU分析50us = 200us
- **总时间 = 900 + 200 = 1100us**

### 【2019年第12题】磁盘容量/SSTF/物理地址计算

**来源**：2019年全国统考（7分）

**题目**：

12.（7分）某计算机系统中的磁盘有300个柱面，每个柱面有10个磁道，每个磁道有200个扇区，扇区大小为512B。文件系统的每个簇包含2个扇区。请回答下列问题。

（1）磁盘的容量是多少？

（2）假设磁头在85号柱面上，此时有4个磁盘访问请求，簇号分别为：100260、60005、101660和110560。若采用最短寻道时间优先（SSTF）调度算法，则系统访问簇的先后次序是什么？

（3）第100530簇在磁盘上的物理地址是什么？将簇号转换成磁盘物理地址的过程是由I/O系统的什么程序完成的？

**参考答案**：

（1）磁盘容量 = 300 x 10 x 200 x 512 / 1024 KB = **3 x 10^5 KB**（约300MB）

（2）每个柱面有 10 x 200 = 2000 个扇区，每个簇含2个扇区，则每个柱面有 1000 个簇。

各簇号对应的柱面号：
- 簇号100260 → 柱面号 = [100260/1000] = **100**
- 簇号60005 → 柱面号 = [60005/1000] = **60**
- 簇号101660 → 柱面号 = [101660/1000] = **101**
- 簇号110560 → 柱面号 = [110560/1000] = **110**

根据SSTF算法（当前磁头在85号柱面），访问顺序按磁头距离从小到大排列：

**100260（柱面100）→ 101660（柱面101）→ 110560（柱面110）→ 60005（柱面60）**

（3）第100530簇在磁盘上的物理地址：

- 柱面号 = [100530 / (10 x 200 / 2)] = [100530 / 1000] = **100**
- 余数 = 100530 % 1000 = 530
- 磁头号 = [530 / (200/2)] = [530 / 100] = **5**
- 扇区号 = (530 x 2) % 200 = 1060 % 200 = **60**

**物理地址**：柱面号100，磁头号5，扇区号60

将簇号转换成磁盘物理地址的过程由**磁盘驱动程序**完成。
